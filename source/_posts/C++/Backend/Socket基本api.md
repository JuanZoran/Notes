# Socket 网络编程准备工作

## 服务器客户端(前端)

-   建立一个 Socket 接口
-   连接服务端
-   向服务端发送数据
-   接受服务端数据
-   关闭 Socket

## 服务器服务端(后端)

-   建立一个 Socket 接口
-   绑定端口
-   监听网络接口
-   等待客户端链接
-   接受客户端数据
-   向客户端返回客户端需要的 数据
-   关闭 Socket
